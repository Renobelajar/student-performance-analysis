# Student Performance Analysis

Python-based student performance analysis and data visualization project using Pandas and Matplotlib.

## Features
- Student score analysis
- Attendance analysis
- Performance statistics
- Data visualization
- Student performance monitoring

## Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset contains:
- Student name
- Class
- Assignment score
- Exam score
- Attendance percentage

## Analysis
- Average student scores
- Best-performing students
- Attendance statistics
- Score comparison
- Student performance trends

## Visualizations
- Bar Chart
- Pie Chart
- Line Chart

## Files
- student-performance-analysis.ipynb
- student_performance.csv

## Concepts Used
- Data Analysis
- Data Visualization
- Statistical Analysis
- Dataset Processing
